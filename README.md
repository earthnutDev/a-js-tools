一个纯函数的工具

## 安装

```sh
npm install   a-js-tools  --save
```

## pure function

- `lmDebounce` 防抖函数
- `lmThrottle` 节流函数
- `getRandomInt` 获取随机的整数
- `getRandomFloat` 获取随机的浮点数

## class 名称转化

- `toLowerCamelCase` Convert hyphenation to hump
- `toSplitCase` Convert small hump to hyphen split
