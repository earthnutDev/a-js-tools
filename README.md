一个纯函数的工具

## 安装

```sh
npm install   a-js-tools  --save
```

## pure function

- `lmDebounce` 防抖函数
- `lmThrottle` 节流函数
- `getRandomInt` 获取随机的整数
- `getRandomFloat` 获取随机的浮点数
- `getRandomString` 获取随机字符串

## class 名称转化

- `toLowerCamelCase` 转化为小驼峰
- `toSplitCase` 转化为连接符分隔
